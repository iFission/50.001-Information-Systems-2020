To import the start code into Android Studio:

i) start AS, create a new Java application module for the problem, e.g., ‘piano’.  you should see that under piano, there is ‘java’ folder, with package ‘com.example’

ii) in Windows explorer / Mac Finder, copy the folders:

Part-B/src/midi
Part-B/src/music
Part-B/src/piano

to 

AndroidStudioProjects/Wk-05/piano/src/main/java

iii) after the copying, you could see the package ‘midi’, ‘music’ and ‘piano’ under the ‘java’ folder.
